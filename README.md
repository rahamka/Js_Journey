# Js_Journey

JS_Journey

A collection of 500+ solved JavaScript problems to master JS step by step.

This repository is designed for beginners and intermediate learners to practice and strengthen their JavaScript skills. It includes problems ranging from basic syntax to advanced concepts and real-world projects.

#Topics Covered

#Tier 1: JavaScript Fundamentals & Syntax (1–150)

Variables, Data Types & Operators
Conditionals & Loops
Functions & Scope
Arrays & Basic Iteration

#Tier 2: Applied JavaScript & DOM Manipulation (151–350)

DOM Selection & Manipulation
Event Handling & Forms
Browser APIs & Storage
Beginner Projects (Calculator, To-do App, etc.)

#Tier 3: Advanced Concepts & Modern Tooling (351–500)

Object-Oriented Programming (OOP) & this
Functional Programming Patterns
Asynchronous JavaScript (Promises, async/await)
Data Structures & Algorithms
Modern JavaScript (ES6+) & Modules
Advanced Projects & Tooling
